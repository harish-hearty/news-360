Overview
News-360 is a web app designed to collect news data from a variety of sources through an API and present it in a streamlined and user-friendly format. It allows users to stay updated with the latest news and headlines conveniently. This project aims to demonstrate the integration of API data into a web application, making it a useful learning resource and a foundation for building more complex applications.

Features
Fetches news data from a third-party API.
Displays news articles and headlines in a user-friendly interface.
Supports filtering and searching for news articles.
Provides a simple and intuitive user experience.
Responsive design for mobile and desktop.
